# brief_1-anno_backend
creazione di un configuratore auto, permette l'accesso ad un User o Admin ad un sito con cui si può configurare l'auto dei propri sogni, si ha la possibilità di personalizzare un auto scegliendo Optional, Accessori Motori compatibili con il modello selezionato e ottenere un preventivo del mezzo. l'Admin avrà accesso ai preventivi fatti dall'utente.
